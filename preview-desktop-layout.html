Desktop Layout Changes vs. Current
① Two-column conditions grid
Weather + Season Banner sit in the left column. Advisory + Frost + Schedules + Priority sit in the right column. On mobile, these revert to a single column.
② Quick Actions as a row
Log Activity and Treatment Calculator sit side by side instead of stacked. Each takes 50% of the content width, making the CTAs feel more like a toolbar.
③ Activity + Stats side by side
Recent Activity and This Month stats sit next to each other. The stat card gains a 3-column number grid (Activities / Rainfall / Last Mow) for a dashboard feel.
